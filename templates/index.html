About the Project

Enhancing image fidelity in low-radiation diagnostic scenarios.

Project Context

This project enhances noisy low-dose PET (Positron Emission Tomography) images using the Restormer deep learning architecture.

By modeling the translation of low-radiation dose noise profiles to paired diagnostic reference quality, the network effectively filters noise layers while conserving clinical boundary structures.

Core Processing Steps

The pipeline functions as a closed loop:

  • Load PET DICOM images using pydicom
  • Normalize image intensity to [0,1]
  • Simulate low-dose PET using Poisson noise
  • Apply MONAI preprocessing (ScaleIntensity, EnsureType)
  • Train Restormer model & generate enhanced PET images
  • Evaluate quality metrics (PSNR, SSIM, RMSE, NRMSE)

Dataset & Architecture Context

Clinical lung cancer slice pipeline configurations.

Dataset: PET DICOM (.dcm) images extracted from a lung cancer PET dataset. Slices are grayscale medical images and are mixed (not separated by cancer class).

Preprocessing Pipeline:

  • Intensity normalization to range [0, 1]
  • Synthetic low-dose generation using Poisson noise
  • MONAI Transforms (ScaleIntensity, EnsureType)
  • Single-channel tensor conversion

Dataset Split:

  • Training: 70%
  • Validation: 15%
  • Testing: 15%

The project uses a lightweight implementation of the Restormer architecture adapted for grayscale PET image enhancement and trained on paired synthetic low-dose and reference PET images.
